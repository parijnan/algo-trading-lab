# Apollo — High-VIX Trend Following Options Strategy

Part of the **Algo Trading Lab** project.

## Strategy Overview

A dual-timeframe Supertrend trend-following options strategy, deployed only when India VIX > 16. Unlike the iron condor (Artemis), Apollo takes a directional position — one side only, in the direction of the trend.

| Parameter | Value |
|---|---|
| Instrument | Nifty weekly options |
| Deploy condition | India VIX > 16 |
| Higher timeframe | 75-min Supertrend (trend regime) |
| Lower timeframe | 15-min Supertrend (entry/exit trigger) |
| Structure | Credit spread (primary) / Debit spread (under test) |
| Exit | 15-min Supertrend flip, index SL, option SL, or spread loss cap |

## Setup

1. Clone the repository
2. Install dependencies: `pip install pandas numpy`
3. Update `BASE_DATA_PATH` in `configs.py` to point to your historical data directory
4. Run precompute: `python precompute.py`
5. Validate output against your charts before proceeding
6. Run backtest: `python backtest.py` (once built)
